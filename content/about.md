# 关于每日新闻聚合

这是一个自动化的新闻聚合博客，每天从各大新闻源收集、汇总并发布最新的新闻内容。

## 项目特点

- **自动化收集**：每日自动从多个新闻源获取最新内容
- **智能汇总**：对新闻进行分类、去重和摘要生成
- **定时发布**：每天固定时间自动更新网站内容
- **开源透明**：所有代码和流程公开可查

## 技术栈

- **博客平台**：Hugo (静态网站生成器)
- **新闻收集**：Python + RSS/API
- **部署**：GitHub Pages
- **自动化**：Cron任务 + GitHub Actions

## 新闻源

本站从以下类型的新闻源收集内容：

1. **主流媒体**：新华社、人民日报、央视新闻等
2. **科技媒体**：36氪、虎嗅、钛媒体等
3. **国际媒体**：BBC、Reuters、AP News等

## 更新频率

- **每日更新**：早上8:00和晚上20:00各更新一次
- **紧急新闻**：重大事件实时更新
